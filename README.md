# GamePluginLauncher

#### 介绍

游戏插件管理器，适用于游戏有多个插件启动，将插件聚合到一起，启动时自行选择

#### 信息

- 主要语言：C#
- 主要技术栈：WPF + Material Design In XAML Toolkit
- .NET版本：.NET6
